Outline of a complete project
1. Proposal a task.
2. Collect data.
3. Clean, analyze, and preprocess data.
4. Formulate the task into a data mining problem. Identify this problem as one type of tasks we have learned in class.
5. Choose multipe algorithms to mine the data.
6. Evaluate the perfomances of these algorithms and choose a best one.
7. Explain and visualize results.
8. Deploy your project in heroku

Project Steps and Due time

Step 1, due 11/15, Wednesday
Tasks: 1) collect data. 2) clean data(noise, outlier, n/a value). 3) apply one algorithm to run the data explain the results.
Submit: 1) code. 2) support file to explain each step and the results. 3) team file to explain each member's work. 4) pandas.ipynb and sklearn.ipynb
