There are lots of terms or algorithms we have not covered yet in this class so far. And this example is just a script of python code. It is also not a complete project. For example, besides the python code, you also need to write a report and to deploy your project in somewhere. Finally, you need to preprare a slides to demo your project in the last meeting of this class. I will update this page frequently next few weeks. Please check this page for new information later

Outline of a complete project
1. Proposal a task.
2. Collect data.
3. Clean, analyze, and preprocess data.
4. Formulate the task into a data mining problem. Identify this problem as one type of tasks we have learned in class.
5. Choose multipe algorithms to mine the data.
6. Evaluate the perfomances of these algorithms and choose a best one.
7. Explain and visualize results.
8. Deploy your project in heroku
